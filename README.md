#TE 566 Accounting Software
Created by Muhammad Rizky Wellyanto 

Project is live at https://te-566-acctg-app.herokuapp.com/

## Setup manual
- make sure you have npm, yarn, nodejs
- run `npm install`
- run `yarn install`
- run `npm start`
- go to http://localhost:3000/
